Phantom & Ghost

Phantom er en OpenBSD server som opprettholder Wireguard VPN og Portal rele for personlig fil-deling. Den er også vert for denne nettsiden. Phantom er 'hardened' så den står bedre imot angrep fra utsiden.

Ghost er en personlig Fedora hjemme-server til personlig bruk. Planlagte programmer er blocky DNS blokker, Jellyfin for streamingg og media-lagring, VaultWarden for passord sikkerhet, PineDocs som kan indekse prosessen, og Homer fungerer som et dashboard for alle prosessene.

Denne nettsiden er et hobbyprosjekt som har som mål å informere andre om hvordan man kan bli tryggere på nettet. Nettsiden har ingen JavaScript eller cookies.
