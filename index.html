Phantom and Ghost

Phantom is an OpenBSD server running Wireguard and a Portal relay for file sharing. It also hosts this website. Phantom is hardened so it's less susceptible to attacks.

Ghost is a Fedora home-server which connects to the Spectre client. Planned usage will be blocky DNS blocker, Jellyfin for media consumption, VaultWarden for password security, PineDocs to catalogue the process and Homer as the dashboard.

This website is a hobby project. The goal is to use none or minimal amounts of JavaScript. The ultimate goal is to use this site to inform more people on the benefits of digital freedom and better internet security.

Operating System

If you're reading this you're probably already using Windows as your operating system (OS). Windows is a great operating system if you're gaming or need support for a certain proprietary software, like Adobe products or the Office Suite.

In general most of us don't really use an OS, we use the apps contained in the OS. If you're just using your OS to browse the web, write in Office or Excel and personal storing photos. All of these things can be done for free and better on Linux distros.

So in a sense Windows is just the environment that you use apps in, so there isn't a good argument or reason not to install Linux, you're already not using Windows as an OS, you're using it as a app environment.

Installing a Linux distro is easier than installing Windows. Linux doesn't require your email or any other social media to function.
